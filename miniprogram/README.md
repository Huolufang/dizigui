# 弟子规学堂 v2.0 微信小程序

## 导入与预览

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 推荐选择仓库根目录 `弟子规学堂/`；根目录 `project.config.json` 已通过 `miniprogramRoot` 指向小程序代码。直接选择 `miniprogram/` 目录也可导入。
4. 没有正式 AppID 时可使用测试号预览；当前配置使用 `touristappid`。

项目不依赖 npm、服务端、云开发或网络请求。课程、原文和 40 讲全部随小程序资源打包。

## 已实现

- 学习、课程、讲稿、复习、我的五个一级入口
- 18 门课程的五步学习轮次、暂停与重学状态
- 90 个原文单元的口头背诵、三级自评和 1/3/7 天排期
- 固定旧课复习快照
- 40 讲完整讲稿、阅读位置和手动已读
- 离线多关键词搜索、分组、命中高亮与段落定位
- 独立反思记录、修改和删除
- 小/中/大三级讲稿字号
- JSON 备份导出、导入前校验、覆盖恢复与失败回滚

18 门课程和 90 个原文单元已由产品所有者按推荐方案确认。规范原文、注释译文映射、讲稿段落、释义差异和读音记录均已固化；正式发布仍需完成微信开发者工具与真机验收。
